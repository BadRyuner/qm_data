- ParameterType: Types of parameters, either exclusive to modifying's types (RangeWeaponTrait) or just data types (Weight: Float)
- ProjectType: Types of projecct: RangeWeapon, MeleeWeapon, Helmet, Armor, Leggings, Boots, Mercenary, MercenaryClass
- ViewType: Display types: Damage, Angle, Percent, Interger, Float0_0, None
- MinValue:	minimum value for modification
- MaxValue:	maximum value for modification
- Step: incrementing value
- TooltipTag: tags for icons of the project parameters

//MAGNUM PROJECT PRICE
- ProjectType: project types
- ItemsGrades: a list of item's id and their respective grade will the item will be included in: example: for range, weapon part will be use from grade 1 until the project reach grade 3, then springs will be included along parts and so on
- ModifyLevelLimit: level limits of project

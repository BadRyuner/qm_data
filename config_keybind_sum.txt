- Layout: for layout binding, UI or Hotkeys
- OtherKeyIdToPress: alternative key's id
- Bind1/2: 1st & 2nd binding, None if no keys assigned
- ControllerBind1/2: 1st & 2nd controller binding, None if no button assigned
- ImmutableBind1/2: only TRUE value accepted, key here cannot be changed
- ImmutableJoystickBind1/2: only TRUE value accepted, joystick option here cannot be changed
- AxisName: Axis ids - Movement, CamMovement, LookAt
- MovementVector: vector x y for movement
- ExclusiveInputMode: can only use this option in either keyboard & mouse or controller
- ForbiddenKeysToBind: cannot use this key to bind
